# Coffee Roasting

Landing page for my current coffee roasting progression.

I will try to update the graphs as timely as possible.

Dubious Sourcing & Unexceptional Roasts Only.

## Software

- [BeansConqueror](https://github.com/graphefruit/Beanconqueror)
- [Artisan](https://github.com/artisan-roaster-scope/artisan)

## Machine

- Smola 300g Fluid Bed Roasting
  - 3mm K-Type Thermocouple for Bean Temp (BT)
  - 4mm K-Type Thermocouple for Environment Temp (ET)
- Mastech M6514
- Random 350g bean cooler/radiator

## Personal Guidelines

| Roast Level      | Weight Loss (WL%) |
| ---------------- | ----------------- |
| Ultra-light (UL) | <10%              |
| Light            | 10 ~ 12.5%        |
| Medium           | 12.5 ~ 15%        |
| Dark             | >15%              |
| Italian Dark     | >18%              |
